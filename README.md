# OOP Delivery (Final exam)

## Overview
A program to track and deliver orders

## Features
### Customer class
- Adds orders
- Has a name, address, and a list of undelivered orders
### Driver class
- Delivers orders
- Has a name, vehicle type, and a list of undelivered orders
### DeliveryOrder class
- Manages an order
- Has the order name, status, and an assigned driver
- Can assign a driver to itself

## How to run
Run the python file "oop_delivery" for an example.
Can also be imported to be used in another file

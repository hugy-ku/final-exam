# OOP Delivery (Final exam)

## Overview
A program that tracks and delivers orders.

## Features
### Customer class
- Has a name, address, and a list of undelivered orders
- Can create orders
### Driver class
- Has a name, vehicle type, and a list of undelivered orders
- Can deliver assigned orders
### DeliveryOrder class
- Has the order name, status, the ordering customer, and an assigned driver
- Can assign a driver

## How to run
Run the python file "oop_delivery" for an example.
Can also be imported to be used in another file.
